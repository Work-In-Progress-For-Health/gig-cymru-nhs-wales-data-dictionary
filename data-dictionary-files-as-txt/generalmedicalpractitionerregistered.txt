This is the code of the GMP with whom the patient is registered.
 
Format: 8 character alpha-numeric
Value
Meaning
GNNNNNNN
A code issued by the Department of Health (DH) for practising GP's in Wales and England. The codes can be viewed on the Health Reference Data Web Pages on HOWIS.
G9999981
No registered GP
 
Note 1: MOD and prison doctors are not GMPs and should not be recorded as Registered GMPs. In this instance use the default code of G9999981' is to be used.
Note 2: When a locum refers, use the code of the GP for whom the locum is acting.
